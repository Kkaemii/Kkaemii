화장품 기획자로 일하며 남성 스킨케어를 올리브영·무신사 2년 연속 1위에 올렸습니다. 아이디어가 실물이 되는 감각이 좋아 개발로 방향을 틀었고, 지금은 같은 방식으로 AI 프로덕트를 만듭니다. 백엔드부터 배포·운영까지 직접 합니다.

**최근에 만든 것**

- 영상 생성 대기 시간을 **5분 → 22초**로 단축
- GraphRAG 챗봇 답변 정확도 **38% → 90%대 후반** (자체 50문항 평가셋)
- "재한 외국인 정보 플랫폼"을 회의에서 제안해 **16개 언어 실서비스**([gltr.life](https://gltr.life))로 런칭
